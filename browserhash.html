BrowserHash.com
Browser Hash uses a few new HTML5 features to compute MD5, SHA-1, and SHA-256 hashes of files in your browser!
This means it's faster, private, and works across all operating systems. I created it because Windows doesn't come with a tool to calculate file hashes.
Browser Hash is my first project to use CoffeeScript instead of JavaScript, Compass to structure and style the project, and Modernizr to detect browser features.

These technologies will be part of the core of my future web apps.

Try out Browser Hash!
